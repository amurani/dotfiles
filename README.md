## dotfiles

### Useage

* Clone repo to `$HOME` i.e. `~/.dotfiles`
* Run `~/.dotfiles/setup/sh && source ~/.zshrc`
* Go do your thing ...

### Other useful tools to make life easier

* [HTTPie](https://httpie.io/) - like `curl` but ...
